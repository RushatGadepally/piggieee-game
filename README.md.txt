# Piggiee Push

A puzzle game prototype created in Unreal Engine 5.

## Gameplay

Push blocks onto their target locations to complete each level.

## Controls

- WASD - Move
- Push blocks by walking into them
- Due to some technical issues the mouse movement is not working as intented in the link that's provided but works completely fine when in game.

## Built With

- Unreal Engine 5
- Blueprint
